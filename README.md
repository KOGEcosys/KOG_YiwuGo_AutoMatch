## 核心功能

- 一页式 KOG Wallet 说明网站
- 多语言 UI 切换与 URL 语言参数
- 互助交换申请入口
- K.NTD / K.USD / K.RMB / RGA / K.TIME / K.FOOD / KOG / K.Gold 多 Token 试算器
- 首次参与者头 1000 K.USD 等值作为 KOG POC / POF 基础贡献
- 地方 80% / KOG 总部 20% 分配
- 参与方 100% KCOINs 等值记录
- 浏览器本地申请资料暂存与 CSV 导出
